

# 📊 Retail Sales Analysis Dashboard (Excel)

## Overview

This project analyzes a retail sales dataset using **Microsoft Excel** to uncover key business insights related to customer behavior, product performance, and regional sales trends.

The analysis converts raw transactional data into an **interactive dashboard** that enables quick monitoring of business performance and supports data-driven decision making.

---

## Dataset

The dataset contains **transaction-level retail sales data** with customer, product, and payment information.

**Key fields included:**

* Transaction ID
* Transaction Date
* Customer ID
* Customer Name
* Product ID
* Product Name
* Product Category
* Quantity
* Unit Price
* Total Amount
* Profit
* Payment Method
* Customer Segment
* Region
* Customer Tenure

The dataset represents multiple sales transactions across different **customers, product categories, and regions**.

---

## Data Preparation

Before performing the analysis, the dataset was prepared using the following steps:

* Verified data consistency and formatting
* Organized structured tabular data
* Created calculated fields for sales and profit metrics
* Generated customer-level metrics for further analysis

---

## Analysis Performed

The analysis focuses on identifying patterns and insights in sales performance.

Key analytical areas include:

* **Sales Performance Analysis**
* **Regional Revenue Distribution**
* **Customer Segment Contribution**
* **Product Category Performance**
* **Payment Method Usage**
* **Customer Value Identification**

---

## Dashboard Features

An interactive Excel dashboard was created to present the insights clearly.

The dashboard includes:

* Total Revenue KPI
* Total Profit KPI
* Average Order Value
* Sales by Region
* Sales by Product Category
* Customer Segment Distribution
* Payment Method Analysis
* Interactive filtering using **Slicers**

These elements allow users to quickly explore and understand sales performance.

---

## Tools & Techniques

The following Excel features were used in this project:

* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Lookup Functions (VLOOKUP / XLOOKUP)
* Basic Data Analysis Techniques

---

## Project Objective

The purpose of this project is to demonstrate how **Excel can be used as an effective data analysis and visualization tool** to transform raw sales data into actionable insights through structured analysis and dashboard design.
